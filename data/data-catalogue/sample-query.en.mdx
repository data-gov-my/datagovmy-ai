## Sample OpenAPI Data Catalogue Queries

The following code is an example of how to make an API query to retrieve the data catalogue mentioned above. You can use different programming languages by switching the code accordingly. For a complete guide on possible query parameters and syntax, please refer to the [Request Query documentation](https://developer.data.gov.my/data-catalogue/request-query). Additionally, you can experiment with the code interactively and send requests at the [Example Requests section](https://developer.data.gov.my/data-catalogue/example-requests).

### Querying in Python

```python
import requests
import pprint

url = "https://api.data.gov.my/data-catalogue?id=elections_federal_ge00_ballots_0&limit=3"

response_json = requests.get(url=url).json()
pprint.pprint(response_json)
```

### Querying in JavaScript

```javascript
var requestOptions = {
  method: "GET",
  redirect: "follow",
};

fetch("https://api.data.gov.my/data-catalogue?id=elections_federal_ge00_ballots_0&limit=3", requestOptions)
  .then((response) => response.json())
  .then((result) => console.log(result))
  .catch((error) => console.log("error", error));
```

### Querying in Swift

```swift
import Foundation
#if canImport(FoundationNetworking)
import FoundationNetworking
#endif

var request = URLRequest(url: URL(string: "https://api.data.gov.my/data-catalogue?id=elections_federal_ge00_ballots_0&limit=3")!,timeoutInterval: Double.infinity)
request.httpMethod = "GET"

let task = URLSession.shared.dataTask(with: request) { data, response, error in
  guard let data = data else {
    print(String(describing: error))
    exit(EXIT_SUCCESS)
  }
  print(String(data: data, encoding: .utf8)!)
  exit(EXIT_SUCCESS)
}

task.resume()
```

### Querying in Dart

```dart
import 'package:http/http.dart' as http;

void main() async {
  var request = http.Request('GET', Uri.parse('https://api.data.gov.my/data-catalogue?id=elections_federal_ge00_ballots_0&limit=3'));

  request.followRedirects = false;

  http.StreamedResponse response = await request.send();

  if (response.statusCode == 200) {
    print(await response.stream.bytesToString());
  }
  else {
    print(response.reasonPhrase);
  }

}
```

### Querying in Kotlin

```kotlin

import okhttp3.RequestBody.Companion.asRequestBody
import java.io.File

```

### Querying in Java

```java
import java.io.*;
import okhttp3.*;
public class Main {
  public static void main(String []args) throws IOException{
    OkHttpClient client = new OkHttpClient().newBuilder()
      .followRedirects(false)
      .build();
    MediaType mediaType = MediaType.parse("text/plain");
    RequestBody body = RequestBody.create(mediaType, "");
    Request request = new Request.Builder()
      .url("https://api.data.gov.my/data-catalogue?id=elections_federal_ge00_ballots_0&limit=3")
      .method("GET", body)
      .build();
    Response response = client.newCall(request).execute();
    System.out.println(response.body().string());
  }
}
```
